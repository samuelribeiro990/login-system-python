# Sistema de Login em Python

Projeto simples desenvolvido para praticar lógica de programação em Python.

## Funcionalidades

- Saudação inicial
- Sistema de login
- Limite de 3 tentativas
- Menu de opções

## Tecnologias

- Python 3

## Como executar

Clone o repositório:

git clone https://github.com/seuusuario/sistema-login-python

Entre na pasta do projeto:

cd sistema-login-python

Execute o programa:

python main.py
